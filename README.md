# How to fetch the metadata of a SPL token


This tutorial aims to explain how to fetch the metadata of a token. To do this, we will present three ways of doing so:
- using the `@metaplex-foundation/js` and `@solana/web3.js` librairies
- using `umi` from [Metaplex](https://developers.metaplex.com/umi)
- using `@solana/kit` librairy and manually decoding the `Metadata Account` info
